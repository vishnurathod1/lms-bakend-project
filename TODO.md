# Task: Configure MySQL Connection with SSL in Django on Windows

## Steps:

1. **Install MySQL Client on Windows**
   - Download and install MySQL Installer from [https://dev.mysql.com/downloads/installer/](https://dev.mysql.com/downloads/installer/)
   - During installation, ensure the "MySQL Client" or "MySQL Command Line Client" feature is selected.
   - Alternatively, install via Chocolatey (if Chocolatey is installed):
     ```
     choco install mysql
     ```
   - Verify installation by running `mysql --version` in a command prompt.

2. **Obtain SSL Certificates**
   - You need the following certificate files for SSL connection:
     - CA Certificate (`ca.pem`)
     - Client Certificate (`client-cert.pem`)
     - Client Key (`client-key.pem`)
   - These files are usually provided by your MySQL server administrator or generated by you if running your own server.
   - Place these files in a secure directory accessible to your Django project.

3. **Update Django `backend/settings.py` Database Configuration**
   - Add SSL options to the `DATABASES` setting under the `'OPTIONS'` key.
   - Example:
     ```python
     DATABASES = {
         'default': {
             'ENGINE': 'django.db.backends.mysql',
             'NAME': 'sandip1_db',
             'USER': 'root',
             'PASSWORD': 'Vishnu@123',
             'HOST': 'localhost',
             'PORT': '3306',
             'OPTIONS': {
                 'ssl': {
                     'ca': '/path/to/ca.pem',
                     'cert': '/path/to/client-cert.pem',
                     'key': '/path/to/client-key.pem',
                     'cipher': 'TLS_AES_128_GCM_SHA256',
                 },
             },
         }
     }
     ```
   - Adjust the paths and SSL parameters according to your certificate files and server requirements.

4. **Ensure PyMySQL Shim**
   - Confirm that `backend/__init__.py` contains:
     ```python
     import pymysql
     pymysql.install_as_MySQLdb()
     ```
   - This is already done.

5. **Test the Connection**
   - Run Django server and verify that you can connect to the MySQL database with SSL enabled.
   - Check for errors related to SSL or connection issues.
